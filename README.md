# krishyogee/homebrew-tap

Homebrew tap for [@krishyogee](https://github.com/krishyogee)'s tools.

## Install

```sh
brew install krishyogee/tap/gitmate
```

## Available formulae

| Formula | Description |
|---------|-------------|
| [gitmate](Formula/gitmate.rb) | Multi-agent AI CLI for Git workflows with approval gates |
